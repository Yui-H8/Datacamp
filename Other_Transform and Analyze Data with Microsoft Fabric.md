# Transform and Analyze Data with Microsoft Fabric
---
**Description**

This course focuses on three key Microsoft Fabric topics: transforming data, exploring and analyzing data, and optimizing performance. Through lessons and interactive exercises, you will learn about different techniques for transforming data in Fabric lakehouses and warehouses, from identifying problems with data, to merging, aggregating, and filtering data. You will also learn about different tools and methods to explore and profile data in Fabric lakehouses and warehouses, and how to monitor and optimize the performance of these Fabric objects.
