# Introduction to Microsoft Fabric
---
**Description**

Welcome to the world of Microsoft Fabric! In this course, you will explore a variety of tools found in the seven Fabric experiences to understand Fabric’s main use cases. Learn how data engineers, data scientists, database administrators, and BI analysts might use Fabric to achieve their business goals. This course was built as a starting point for preparing for Microsoft’s Fabric Analytics Engineer Associate certification (DP-600).
